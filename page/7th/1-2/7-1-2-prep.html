التحضير


مكوّنات الحاسوب المادية :
- أجزاء ملموسة (Hardware) تعمل مع البرمجيات (Software) لتنفيذ الأوامر.

أنواع المكوّنات :
- مكوّنات خارجية (Peripherals): وحدات إدخال + وحدات إخراج.
- مكوّنات داخلية (Internal Components): داخل وحدة النظام (System Unit).

المكوّنات الخارجية :
- وحدات الإدخال: لوحة المفاتيح، الفأرة، الماسح الضوئي، الكاميرا، الميكروفون.
- وحدات الإخراج: الشاشة، الطابعة، السماعات، البروجيكتور.

المكوّنات الداخلية :
- اللوحة الأم (Motherboard): تربط جميع المكونات.
- وحدة المعالجة المركزية (CPU): دماغ الحاسوب.
- الذاكرة (RAM/ROM): RAM للتخزين المؤقت، ROM للتشغيل الأساسي.
- القرص الصلب (HDD): تخزين طويل المدى.
- بطاقة الرسوم (GPU): عرض الصور والفيديو.
- مزود الطاقة (Power Supply): يزوّد المكونات بالكهرباء.
- مروحة التبريد (Cooling Fan): تمنع ارتفاع الحرارة.

التكامل الوظيفي :
- جميع المكوّنات تعمل معاً (داخلية + خارجية) لأداء المهام بكفاءة.
