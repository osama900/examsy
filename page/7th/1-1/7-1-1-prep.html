الدرس الأول - جهاز الحاسوب 💻

التحضير


جهاز الحاسوب :
- جهاز إلكتروني يستقبل البيانات ويعالجها ويخزنها ويُخرجها كمعلومات مفيدة.

مبدأ عمل الحاسوب :
- الإدخال (Input): إدخال البيانات عبر لوحة المفاتيح، الفأرة، الكاميرا...
- المعالجة (Processing): CPU يعالج البيانات بالعمليات الحسابية والمنطقية.
- التخزين (Storage): حفظ البيانات والمعلومات لاستخدامها لاحقاً.
- الإخراج (Output): عرض النتائج عبر الشاشة، الطابعة، السماعة.

البيانات والمعلومات :
- البيانات: حقائق أولية غير معالجة (أرقام، كلمات، رموز، أصوات).
- المعلومات: بيانات تمت معالجتها فأصبحت مفهومة ومفيدة.

أنواع الحواسيب :
- الحاسوب المكتبي: ثابت، قوي الأداء، مساحة تخزين كبيرة.
- الحاسوب المحمول (Laptop): خفيف وسهل النقل.
- الحاسوب اللوحي (Tablet): شاشة لمس، خفيف، بدون لوحة مفاتيح مادية.
- الهاتف الذكي (Smartphone): يجمع بين الهاتف والحاسوب، يشغّل التطبيقات.

استخدامات الحاسوب :
- التعليم: التعلم الإلكتروني والبحث.
- الصحة: تشخيص الأمراض وإدارة الملفات الطبية.
- التجارة: البيع والشراء عبر الإنترنت.
- النقل: تتبع الشحنات باستخدام GPS.
- الرياضة: تحليل أداء الرياضيين.
- الترفيه: الألعاب والأفلام ووسائل التواصل.
