الدرس الثالث - صيانه الاعطال البرمجية 🧰

التحضير


أعطال البرمجيات الشائعة :
1- بطء الجهاز (قلة RAM، امتلاء القرص، فيروسات).
2- توقف البرامج عن الاستجابة.
3- بطء تشغيل برنامج معين.


إجراءات الصيانة الدورية :
1- تحديث نظام التشغيل Windows.
2- استخدام أدوات النظام (Troubleshoot, Disk Cleanup, Defragment).
3- إزالة البرامج غير المستخدمة.
4- النسخ الاحتياطي (محلي / سحابي).
5- إنشاء نقطة استعادة النظام.


أداة POST (power on self test ) :
1- تفحص مكونات الجهاز عند بدء التشغيل.
2- تصدر أصواتًا أو رسائل خطأ تدل على الأعطال.


حلول عملية للمشكلات :
1- فرض إنهاء البرامج عبر (Task Manager).
2- إعادة تشغيل الجهاز.
3- تحديث البرنامج أو إعادة تثبيته.
4- فحص المكونات المادية عند الحاجة.
